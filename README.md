## Installation
To run this portfolio locally, follow these steps:

1. **Clone the repository:**
    ```bash
    git clone https://github.com/Youbey/portfolio.git
    ```
2. **Navigate to the project directory:**
    ```bash
    cd portfolio
    ```
3. **Open `index.html` in your web browser:**
    ```bash
    open index.html
    ```
